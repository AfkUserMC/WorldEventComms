**WorldEventComms** is a simple client mod for Wynncraft that adds commands to help you communicate your tracked world event to the rest of the party to make sure everyone is on the same page.

The commands that can be used are below:
- /we next: Sends your currently tracked world event to party chat.
- /we busy: Sends information about the ongoing world event to party chat.
- /we none: Disappoint everyone by telling them there's no active world events.

Partially compatible with Wynntils, almost.
